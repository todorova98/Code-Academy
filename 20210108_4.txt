4 зад Да се преобразуват от двоична в десетична бройна система
дадените числа, като се използва формулата за представяне:
а) 100011 б) 1000000 в) 1101001 г) 1001


а) 100011 = 1*2^5 + 0*2^4 + 0*2^3 + 0*2^2 + 1*2^1 + 1*2^0 = 32 + 2 + 1 = 35 => 100011(2) = 35(10)


б) 1000000 = 1*2^6 + 0*2^5 + 0*2^4 + 0*2^3 + 0*2^2 + 0*2^1 + 0*2^0 = 64

в) 1101001 = 1*2^6 + 1*2^5 + 0*2^4 + 1*2^3 + 0*2^2 + 0*2^1 + 1*2^0 = 64 +32 + 8 + 1 = 105

г) 1001 = 1*2^3 + 0*2^2 + 0*2^1 + 1*2^0 = 9
